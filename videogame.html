2D Video Game Development

A 2D game project focused on enemy AI behavior, projectile systems, and gameplay polish.

Overview

This project is a custom-built 2D game developed from the ground up. The primary focus was on implementing responsive enemy behavior, engaging combat mechanics, and a scalable architecture for adding new content.

Gameplay Systems

  • Enemy AI with movement patterns and attack phases
  • Projectile and collision systems
  • Player movement and combat mechanics
  • Scene-based progression

Tech Stack

  • Unity (2D)
  • C# scripting
  • Custom enemy behaviors
  • Sprite-based animation

Highlights

  • Enemies dynamically adjust behavior based on player position
  • Shotgun-style projectile patterns
  • Expandable architecture for new enemy types
