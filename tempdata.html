Temperature & Population Data Analysis

An interactive web-based data analysis platform that combines temperature and population statistics across cities, featuring dynamic visualizations and comprehensive filtering capabilities powered by a MySQL database and Python data pipeline.

View Live Project

Overview

This project provides a comprehensive analysis tool for exploring temperature and population data across multiple cities spanning from 1950 to 2021. Built as a collaborative effort, the system combines real-world climate data with demographic information, allowing users to visualize trends, compare cities, and identify patterns related to temperature changes and population growth over decades.

Key Features

  • Interactive charts comparing yearly average temperatures across cities
  • Population and temperature correlation analysis and visualization
  • Historical population data spanning 72 years (1950-2021)
  • Climate extremes tracking: count of unlivable days (temperatures exceeding 95°F)
  • Dynamic filtering system allowing users to select and compare multiple cities
  • Real-time data rendering with responsive chart visualizations
  • Comprehensive data-driven insights into climate and demographic trends

Tech Stack

Backend

MySQL database for data storage

Data Pipeline

Python for web scraping & ETL

Frontend

PHP for server-side rendering

Visualization

CanvasJS for interactive charts

Data Collection & Processing

  • Automated Python script for web scraping temperature data from online sources
  • Data normalization and validation before database insertion
  • MySQL database design for efficient querying and aggregation
  • Historical data spanning multiple decades with consistent formatting
  • Support for multi-city comparisons and cross-year analysis
  • Automated data pipeline for periodic updates and maintenance

Core Functionality

  • Compare yearly average temperatures between selected cities with interactive charts
  • Analyze population growth patterns and trends over time
  • Correlate temperature changes with population demographics
  • Track extreme weather events and their frequency across different cities
  • Filter and sort data by multiple criteria for custom analysis
  • Export and view detailed statistics for academic and research purposes

Technical Highlights
